【学习笔记】多项式全家桶

前言

本文略去多项式的定义，毕竟初中数学都讲过。

本文会出现一些可能没有见过的表达：

  • 对于一个关于 \(x\) 的多项式 \(f(x)\)，有时也称其最高次项次数为 \(f(x)\) 的度数。
  • 多项式同样定义了四则运算，包括常数与多项式的四则运算和多项式之间的四则运算，同时也衍生出了同余符号 \(\equiv\)、对数函数 \(\ln(f(x))\)，指数函数 \(\exp(f(x))\) 等等。
  • 下文一般采用 \(f(x)~\bmod x^n\) 来表示多项式 \(f(x)\) 的所有次数小于 \(n\) 的项之和。
  • 下文有时会采用 \([x^k]f(x)\) 表示多项式次数为 \(k\) 的项。

注意：本文的知识环环相扣，请勿乱序阅读。

拉格朗日插值

插值，通俗地讲，就是指通过多项式在某些点的值求出整个多项式的表达式。

而拉格朗日插值就是计算插值的一种方法，它支持通过 \(n + 1\) 个点插出一个度数为 \(n\) 的多项式。

按惯例，献上模板题。

首先对于多项式 \(f(x)\)，必然满足 \(f(x) \equiv f(a)~(\bmod (x - a))\)。

证明：

转化为 \(f(x) - f(a) \equiv 0~(\bmod~(x - a))\)。

设左侧的多项式为 \(g(x)\)，显然其常数项为 \(0\)。

又因为其第 \(i\) 项为 \(p_i(x^i - a^i)\)（\(p_i\) 为系数），并且 \(x^i - a^i\) 必定可以分解为 \((x - a) \cdot q(x)\) 的形式（\(q(x)\) 为另一个多项式），所以得证。

所以我们可以得到一个关于 \(f(x)\) 线性同余方程组： \[ \left\{ \begin{matrix} f(x) \equiv y_1~(\bmod (x - x_1)) \\ f(x) \equiv y_2~(\bmod (x - x_2)) \\ f(x) \equiv y_3~(\bmod (x - x_3)) \\ \vdots \\ f(x) \equiv y_n~(\bmod (x - x_n)) \end{matrix} \right. \] 这里可以使用中国剩余定理解方程组（这里就没有必要关注它们是否互质了）。

我们可以设 \(M = \prod_{i = 1}^n (x - x_i)\)，\(M_i = \frac{M}{(x - x_i)} = \prod_{j \ne i} (x - x_j)\)。

则 \(M_i\) 在模 \(x - x_i\) 意义下的逆元为 \(\prod_{j \ne i}\frac{1}{x_i - x_j}\)。

（因为 \(M_i M_i^{-1} = \prod_{j \ne i}\frac{x - x_j}{x_i - x_j} \equiv 1~(\bmod (x - x_i))\)，则对于每一个 \(j \ne i\) 都存在 $ x - x_j x_i - x_j~((x - x_i))$，移项可得 \(x - x_i \equiv 0~(\bmod (x - x_i))\)，这显然是成立的。）

所以有 \(f(x) = \sum_{i = 1}^n y_i\prod_{j \ne i} \frac{x - x_j}{x_i - x_j}~(\bmod~M)\)，它在模意义下是唯一的。

（实际上有另外的证明方式证明了其在通常意义下也是唯一的，这里略去。）

本题中，我们只需要代入 \(k\) 求解即可，时间复杂度为 \(\mathcal{O}(n^2)\)。不过更多时候，我们需要求出其所有系数，也可以做到 \(\mathcal{O}(n^2)\)，甚至还存在更低的复杂度，这里就不展开介绍了。

多项式乘法

多项式乘法的定义，这里就不必多言了，初中数学都讲过。

代数形式：

设有多项式 \(f(x) = a_0 + a_1 x+ a_2x^2 + \cdots+ a_nx_n\)，\(g(x) = b_0 + b_1 x+ b_2x^2 + \cdots+ b_mx_m\)。

则它们的乘法积为 \(t(x) = c_0 + c_1 x + c_2 x^2 + \cdots +c_{n + m}x^{n + m}\)。

其中，对于 \(t(x)\) 的系数 \(c_k\) 有公式：\(c_k = \sum_{i = 0}^k a_i b_{k - i}\)。

可以发现这个公式的形式十分类似卷积。但请注意：它们不是同一个东西，不要弄混淆了。

事实上，多项式乘法就是用于求卷积的。如果您对卷积感兴趣，建议百度。

现在，给出两个多项式 \(f(x)\) 和 \(g(x)\) 的系数，求出它们的乘法积 \(t(x)\)。\(f(x)\) 和 \(g(x)\) 的次数小于 \(10^5\)。

根据上面的公式，我们可以得到一个 \(\mathcal{O}(n^2)\) 的优秀算法。

我们考虑换一种思想计算这个乘法积。

众所周知，有这样一条著名的定理：任意 \(k\) 个点可以唯一确定一个至多为 \(k - 1\) 次的多项式。

所以我们得到了一个新的思路：先求出足够多的 \(f(x)\) 和 \(g(x)\) 的点值，将它们用 \(\mathcal{O}(n)\) 的复杂度乘起来（横坐标不变，纵坐标相乘），再将这些点转化为一个多项式（上面提到了这样的过程叫做插值）。

这种方法叫做离散傅里叶变换（即著名的 \(\operatorname{DFT}\)）。

问题是，如果钦定一些 \(x\) 去求对应的 \(f(x)\) 的值，时间复杂度还是 \(\mathcal{O}(n^2)\) 的。

所以，下面介绍快速傅里叶变换，同时会介绍快速数论变换（即著名的 \(\operatorname{FFT}\) 和 \(\operatorname{NTT}\)）。

前置知识：单位根，原根。

还记得二次函数 \(y = x^2\) 的性质吗？

它关于 \(y\) 轴对称，所以我们如果代入 \(x = k\)，同样可以得到 \(x = -k\) 时 \(y\) 的值。

这启发我们，可以选取一些特殊的 \(x\)，使得我们可以在求出一些点值的同时，快速地求出另外的一些点值。

这个时候，单位根就派上用场了。

先讲一些使得单位根能够应用于此的性质：

  • 记 \(\omega_n\) 为 \(n\) 阶单位根，即满足 \(\omega_n^k = e^{\frac{2k\pi i}{n}}\)，而 \(e^{\frac{2k\pi i}{n}}=\cos \frac{2k\pi}{n} + i \sin \frac{2k\pi}{n}\)，\(i\) 为虚数单位。

（注意 \(n\) 次单位根的 \(n\) 一般取 \(2\) 的正整数次幂，而下文皆如此。）

这个计算公式不仅可以让我们计算出单位根的具体数值，并且可以得出以下性质：

  • \(\omega_n^n = \cos 2\pi + i \sin 2\pi = 1 = \omega_n^0\)。

这条性质说明 \(n\) 次单位根存在长度为 \(n\) 的指数循环节；

  • \(\omega_n^{\frac{n}{2}} = \cos \pi + i \sin \pi= -1\)。

这条性质间接地证明了 \(n\) 次单位根的最小指数循环节长度为 \(n\)，同时为下文折半引理的证明做了铺垫。

  • 单位根消去引理：\(\omega_{2n}^{2k} = \omega_n^k\)。

根据公式有 \(\omega_{2n}^{2k}= e^{\frac{4k\pi i}{2n}} = e^{\frac{2k\pi i}{n}} = \omega_{n}^k\)，得证。

这条性质可以做为沟通不同次单位根的桥梁，同时也证明了 \(n\) 次单位根的值域是 \(2n\) 次单位根值域的真子集。

  • 单位根折半引理：\(\omega_n^{k+\frac{n}{2}} = -\omega_{n}^k\)。

根据上面的性质有 \(\omega_n^{k + \frac{n}{2}} = \omega_n^k \cdot \omega_n^{\frac{n}{2}} = -\omega_n^k\)，得证。

这条性质可以做为沟通同次不同指数的单位根的桥梁，为 \(\operatorname{DFT}\) 的分治优化创造了可能。

而我们该如何运用单位根的这些性质呢？

我们将 \(f(x) = a_0 + a_1 x + a_2 x^2 + a_3x^3\) 做一个小小的改动： \[ \begin{aligned} f(x) &= a_0 + a_1 x + a_2 x^2 + a_3x^3 \\ &=a_0 + a_2 x^2 + a_1 x + a_3x^3 \\ &= (a_0 + a_2 x^2) + x(a_1 + a_3 x^2) \\ &= f_1(x^2) + xf_2(x^2) \end{aligned} \] 可以发现，这样的变换，会使得原多项式被拆分为两个规模缩小了一半多项式。

所以，我们又有了一个新的算法模型：

我们考虑将 \(f(x)\) 拆分为两个规模缩小了一半的多项式 \(f_1(x), f_2(x)\)，然后对它们进行分治，接着，我们可以用 \(f_1(x),f_2(x)\) 在 \(x = k^2\) 处的点值计算出 \(f(x)\) 在 \(x = k\) 处的点值。如果分治时每一层的复杂度都能够做到 \(\mathcal{O}(n)\)，那么总的时间复杂度就是 \(\mathcal{O}(n \log_2 n)\)。

接下来，我们考虑利用单位根的性质，将每一层转化点值的操作复杂度变为 \(\mathcal{O}(n)\)。

如果我们将单位根 \(\omega_{n}^k\) 和 \(\omega_n^{k + \frac{n}{2}}\) 分别代入多项式 \(f(x)\)： \[ \begin{matrix} f(\omega_{n}^{k}) = f_1(\omega_n^{2k}) + \omega_n^kf_2(\omega_n^{2k}) = f_1(\omega_{\frac{n}{2}}^{k}) + \omega_n^kf_2(\omega_{\frac{n}{2}}^{k}) \\ f(\omega_{n}^{k + \frac{n}{2}}) = f_1(\omega_n^{2k + n}) + \omega_n^{k + \frac{n}{2}}f_2(\omega_n^{2k + n}) = f_1(\omega_{\frac{n}{2}}^{k}) - \omega_n^kf_2(\omega_{\frac{n}{2}}^{k}) \end{matrix} \] 可以发现，只要我们求出了 \(f_1(x),f_2(x)\) 在 \(x = \omega_{\frac{n}{2}}^k\) 处的点值，我们就可以直接得出 \(f(x)\) 在 \(x = \omega_n^k,x = \omega_n^{k + \frac{n}{2}}\) 两处的点值！

具体来说，我们只要在所有长度为 \(n\) 的多项式中求出在所有 \(n\) 次单位根处的点值即可。

这样，每一层做到 \(\mathcal{O}(n)\) 的复杂度已经易如反掌了。

既然无法解释，那就考虑推式子证明吧：

设 \(y_{k}\) 是 \(f(x),g(x)\) 在傅里叶变换后，在 \(x = \omega_p^k\) 处点值相乘的结果。

我们设多项式 \(s(x) = y_0 + y_1 x + y_2 x^2 + \cdots + y_{p - 1} x ^{p - 1}\) 在 \(x = \omega_n^{0}, \omega_n^{-1}, \cdots\cdots,\omega_n^{-(p - 1)}\) 处的点值分别为 \(u_{0 \sim p - 1}\)，即满足： \[ u_k = \sum_{i = 0}^{p - 1} y_i(\omega_p^{-k})^i \] 现在我们要用它证明等式 \(c_k = \frac{u_k}{p}\) 成立，其中 \(c_k\) 指最终 \(f(x)\) 和 \(g(x)\) 的乘法积的系数。

我们直接莽式子： \[ \begin{aligned} u_k &= \sum_{i = 0}^{p - 1} y_i(\omega_p^{-k})^i \\ &= \sum_{i = 0}^{p - 1} (\sum_{j = 0}^{p - 1}c_j(\omega_p^i)^j)(\omega_p^{-k})^i \\ &= \sum_{i = 0}^{p - 1} \sum_{j = 0}^{p - 1}c_j(\omega_p^{j - k})^i \\ &= \sum_{j = 0}^{p - 1}c_j \sum_{i = 0}^{p - 1} (\omega_p^{j - k})^i \end{aligned} \] 然后对 \(\sum_{i = 0}^{p - 1} (\omega_p^{j - k})^i\) 这一部分分讨，设 \(v = j - k\)，则该式等于 \(\sum_{i = 0}^{p - 1} (\omega_p^v)^i\)。

在这个式子的意义下，这就是一个等比数列求和。

如果公比 \(\omega_p^v = 1\)，即 \(v = 0\)，则此时原式等于 \(p\)；

否则，套用等比数列公式，原式等于： \[ \begin{aligned} &\frac{(\omega^v)^p - 1}{\omega_p^v - 1} \\ =&\frac{(\omega^p)^v - 1}{\omega_p^v - 1} \\ =&\frac{1 - 1}{\omega_p^v - 1} \\ =& 0 \end{aligned} \] 故当且仅当 \(v = j - k = 0\)，即 \(j = k\) 时，后面的这一部分才会产生贡献，则有 \(u_k = pc_k\)，即 \(c_k = \frac{1}{p}u_k\)。

得证。

（这里提一句，上面的证明本质上是单位根反演，感兴趣的读者可以自行查阅相关资料。）

这样，算法真正大功告成啦！

实际上，\(\operatorname{IFFT}\) 还有一种写法：

也就是把整个点值序列除了 \(0\) 之外的位置翻转过来，然后做和 \(\operatorname{FFT}\) 一样的变换。

接下来证明其正确性：

我们依旧考虑证明 \(c_k = \frac{u_k}{p}\)，条件是这个等式：

\[ u_k = y_0+\sum_{i = 1}^{p - 1} y_{p - i}(\omega_p^{k})^i \]

我们继续莽式子：

\[ \begin{aligned} u_k &= y_0+\sum_{i = 1}^{p - 1} y_{p - i}(\omega_p^{k})^i \\ &= \sum_{i = 0}^{p - 1} y_i(\omega_p^{k})^{p - i} \\ &= \sum_{i = 0}^{p - 1} y_i(\omega_p^{k})^{p - i} \\ &= \sum_{i = 0}^{p - 1} y_i(\omega_p^{k})^{-i} \\ &= \sum_{i = 0}^{p - 1} y_i(\omega_p^{-k})^{i} \end{aligned} \]

然后就是和上面一模一样的情况了。

当然，\(\operatorname{FFT}\) 还存在很大的优化空间：

  • 减小变换次数

可以发现，对于一个复数 \(a + bi\)，有 \((a + bi)^2 = (a^2 - b^2) + 2abi\)。

如果我们把 \(g(x)\) 放到 \(f(x)\) 的虚部上，对 \(f(x)\) 跑一次 \(\operatorname{FFT}\)，然后将 \(f(x)\) 的每个点值变为其平方。那么，虚部除以 \(2\) 不就是我们想要的值吗？

这样，我们就可以只跑一遍 \(\operatorname{FFT}\)，一遍 \(\operatorname{IFFT}\) 即可，常数优化至原来的 \(\frac{2}{3}\)。

  • 预处理单位根

首先我们都知道单位根的消去引理：\(\omega_{2n}^{2k} = \omega_n^k\)。

在 \(\operatorname{FFT}\) 的过程中，枚举的 \(n\) 是逐级倍增的，也就是说，我们用到的单位根会越来越多，并且每一阶单位根的每一个都会被用到。

由消去引理可以发现， \(n\) 阶单位根的第 \(k\) 个，实际上就等于 \(2n\) 阶单位根的第 \(2k\) 个。

所以，每当 \(n\) 倍增为 \(2n\)，会有 \(n\) 个单位根是已经用到过的，剩下的 \(n\) 个单位根则是新出现的。

如果把这个过程反过来看，即 \(2n\) 变为 \(n\)，那么可以发现：所有 \(n\) 阶单位根都被包含在 \(2n\) 阶单位根中！

综上，我们对于最大的 \(n\)，把所有 \(n\) 阶单位根预处理出来，做 \(\operatorname{FFT}\) 时就不需要每一层都重新处理一遍单位根了。这不仅大大减小了常数，貌似还会提高一些精度。

然而，\(\operatorname{FFT}\) 的精度仍然是一个令人头疼的问题。我们想考虑一种方式，使得整个运算建立在整数域上。

整数，想到数论，有一个东西恰好符合其所有要求——原根。

关于原根的定义和性质，可以参考我的博客《整除和同余相关理论》。

可以发现，使得单位根可用于 \(\operatorname{FFT}\) 的性质，原根同样满足：

  • \(n\) 次原根计算公式为 \(g_n^k = g^{\frac{k(p - 1)}{n}} \bmod~p\)，其中 \(p\) 为模数，\(g\) 特指模 \(p\) 的最小原根。

这里的 \(p\) 一般取 \(998244353\)，它的最小原根等于 \(3\)，并且有 \(998244353 = 2^{23}\times 7\times 17 + 1\)，其中 \(2^{23} = 8388608\)，其大小完全可以满足我们的需求。

同样，我们也可以得出 \(g_n^n = g_n^0 \equiv 1~(\bmod ~p)\) 和 \(g_n^{\frac{n}{2}} \equiv -1~(\bmod ~p)\) 以及它们的推论。

  • 原根同样存在消去引理和折半引理。

所以，在 \(\operatorname{FFT}\) 中对多项式的变换，在 \(\operatorname{NTT}\) 中同样适用。同时，也需要 \(\operatorname{Rader}\) 排序。

同理，对于 \(\operatorname{INTT}\)，原根同样可以在原来的基础上求倒数再乘 \(\frac{1}{n}\)，只需要求逆元即可。

到这里，正文才刚刚开始。。。

首先我们要知道一点求导和积分的前置知识。

对于要用到的结论，这里就直接摆出来了： \[ \begin{aligned} & f(x) = e^x \Rightarrow f'(x) = e^x \\ & f(x) = \operatorname{ln} x \Rightarrow f'(x) = \frac{1}{x} \\ & f(x) = ax^b \Rightarrow f'(x) = abx^{b - 1} \\ & (f(x) \pm g(x))' = f'(x) \pm g'(x) \\ & (f(g(x))' = f'(g(x)) * g'(x) \end{aligned} \] 其中 \('\) 表示求导。

所以我们可以推出多项式求导公式： \[ f(x) = \sum_{i = 0}^n a_i x^i \Rightarrow f'(x) = \sum_{i = 0}^{n - 1}(i + 1)a_{i + 1} x^i \] 当然，如果知道了多项式的导数，我们也可以用积分求出原多项式： \[ f'(x) = \sum_{i = 0}^na_ix^i \Rightarrow f(x) = \sum_{i = 1}^{n + 1}\frac{a_{i - 1}x^i}{i} \] 注意：多项式积分是不能得到正确的常数项的。不过大部分题目在数据上做了保证，可以直接认为其为 \(0\)。

这两个过程的代码写起来就很简单了，时间复杂度都是 \(\mathcal{O}(n)\) 的：

求导：（把度数为 \(n-1\) 的多项式 \(f(x)\) 求导，结果为 \(g(x)\)）

积分：（把多项式 \(f(x)\) 求积分，结果为度数为 \(n-1\) 的多项式 \(g(x)\)）

在文章后面的部分，有很多地方会把一些式子通过求导变成方便计算的形式，然后再用积分求出答案。

以上结论已经能够满足大部分 \(\operatorname{OI}\) 题目的需求了。

多项式乘法逆元

给出一个度数为 \(n - 1\) 的多项式 \(f(x)\)，求出一个度数同为 \(n - 1\) 多项式 \(g(x)\)，满足 \(f(x)*g(x) \equiv 1~(\bmod x^n)\)。

其中 \(*\) 表示多项式乘法。系数对 \(998244353\) 取模。

先给出算法：倍增。

假如我们已经求出了一个多项式 \(t\)，满足 \(f(x) * t(x) \equiv 1~(\bmod~x^{\lceil\frac{n}{2}\rceil})\)。

又因为 \(f(x)*g(x)\equiv 1~(\bmod x^n)\)，说明其前 \(n\) 项除常数项为 \(1\) 之外其余项均为 \(0\)。

那么显然有 \(f(x) * g(x) \equiv 1~(\bmod~x^{\lceil\frac{n}{2}\rceil})\)。

所以 \(f(x) * (g(x) - t(x)) \equiv 0~(\bmod~x^{\lceil\frac{n}{2}\rceil})\)。

\(f(x)\) 由于要求逆元，所以不可能为 \(0\)，那么有 \(g(x) - t(x) \equiv 0~(\bmod~x^{\lceil\frac{n}{2}\rceil})\)。

然后推一波式子：

\[ \begin{matrix} g^2(x) \equiv 2 * g(x) * t(x) - t^2(x)~(\bmod~x^{n}) \\ \Downarrow \\ f(x) * g^2(x) \equiv 2 * f(x) * g(x) * t(x) - f(x) * t^2(x)~(\bmod~x^{n}) \\ \Downarrow \\ g(x) \equiv 2 * t(x) - f(x) * t^2(x)~(\bmod~x^{n}) \end{matrix} \]

我们知道 \(f(x)\) 和 \(t(x)\)，那么自然也可以求出 \(g(x)\) 了。

对于常数项，直接求出其逆元即可。

时间复杂度 \(\mathcal{O}(n \log_2n)\)。

多项式除法

（话说多项式除法的地位和多项式乘法比起来为何如此卑微。。。）

给出一个度数为 \(n - 1\) 的多项式 \(f(x)\) 和一个度数为 \(m - 1\) 多项式 \(g(x)\)，请求出多项式 \(q(x)\) 和 \(r(x)\)。

满足：

  • \(f(x) = g(x) * q(x) + r(x)\)；
  • \(q(x)\) 度数为 \(n - m\)，\(r(x)\) 度数小于 \(m - 1\)；

所有系数对 \(998244353\) 取模。

没记错的话，初中数学老师讲过一种 “大除法”。

不过本题是在模意义下计算的，所以说这种方法不是很适用。

接下来，我们需要考虑用先前学过的知识

那么这操作就很关键了：系数翻转。

（为了方便，下文设 \(f(x)\) 度数为 \(n\)，设 \(g(x)\) 度数为 \(m\)，其余的度数变化同理。）

我们设对 \(f(x)\) 的第 \(0\sim n\) 个系数翻转后为 \(f^{r(n)}(x)\)。

（注意下文不会刻意地区分 \(f^{r(n)}(x)\) 中 \(r(n)\) 和余数 \(r(x)\) 的区别。）

那么可以发现有： \[ f^{r(n)}(x) = \sum_{i = 0}^{n} a_{n - i} x^i \\ \Downarrow \\ f^{r(n)}(x) = x^{n} f(\frac{1}{x}) \] 又因为 \(f(\frac{1}{x}) = g(\frac{1}{x}) * q(\frac{1}{x}) + r(\frac{1}{x})\)，所以： \[ \begin{aligned} f^{r(n)}(x) &= x^n * (g(x) * q(x) + r(x)) \\ &= x^n * (g(x) * q(x)) + x^n r(x) \\ &= x^m * g(x) * x^{n - m} * q(x) + x^nr(x) \\ &= g^{r(m)}(x) * q^{r(n - m)}(x) + r^{r(n)}(x) \end{aligned} \] 然后，既然 \(r(x)\) 的度数小于 \(m\)，那么翻转过后，\(r^{r(n)}(x)\) 的前 \(n - m\) 个系数就必然为 \(0\)。

所以，如果我们在 \(\bmod~x^{n - m + 1}\) 意义下做上述运算，不仅可以去掉余数 \(r(x)\) 的影响，同时我们还可以发现，这样完美地保留了 \(q(x)\)。

那么有： \[ \begin{matrix} f^{r(n)}(x) \equiv g^{r(m)}(x) * q^{r(n - m)}(x)~(\bmod~x^{n - m + 1}) \\ \Downarrow \\ q^{r(n - m)}(x) \equiv \frac{g^{r(m)}(x)}{f^{r(n)}(x)} ~(\bmod~x^{n - m + 1}) \\ \Downarrow \\ q(x) = (\frac{g^{r(m)}(x)}{f^{r(n)}(x)})^{r(n - m)} ~(\bmod~x^{n - m + 1}) \end{matrix} \] 至于余数，我们直接把 \(g(x)\) 和 \(q(x)\) 乘起来，再用 \(f(x)\) 减去即可。

多项式开根

给出一个度数为 \(n - 1\) 的多项式 \(f(x)\)，求出一个度数同为 \(n - 1\) 多项式 \(g(x)\)，满足 \(g^2(x) \equiv f(x)~(\bmod x^n)\)。

保证有 \([x^0]f(x) = 1\)。系数对 \(998244353\) 取模。

我们还是用倍增法来做。

临界条件：\(n = 1\) 时，\([x^0]g(x) = 1\)。

设已经求出了一个多项式 \(t(x)\)，满足 \(t^2(x) \equiv f(x~(\bmod~x^{\lceil\frac{n}{2}\rceil})\)。

同时设 \(g(x)^2\equiv f(x)~(\bmod x^n)\)。

然后又是一波推式子： \[ \begin{aligned} & t^2(x) \equiv f(x)~(\bmod~x^{\lceil\frac{n}{2}\rceil}) \\ & t^2(x) - f(x)\equiv 0~(\bmod~x^{\lceil\frac{n}{2}\rceil}) \\ & (t^2(x) - f(x))^2 \equiv 0~(\bmod~x^n) \\ & (t^2(x) + f(x))^2 \equiv 4* t^2(x) * f(x)~(\bmod~x^n) \\ & (\frac{t^2(x) + f(x)}{2*t(x)})^2 \equiv f(x)~(\bmod~x^n) \\ & \frac{t^2(x) + f(x)}{2*t(x)} \equiv g(x)~(\bmod~x^n) \\ & \frac{t(x)}{2} + \frac{f(x)}{2 * t(x)} \equiv g(x)~(\bmod~x^n) \end{aligned} \] 推到这里式子就好做了。时间复杂度 \(\mathcal{O}(n \log_2n)\)。

多项式对数函数

给出一个度数为 \(n - 1\) 的多项式 \(f(x)\)，求出一个度数同为 \(n - 1\) 多项式 \(g(x)\)，满足 \(g(x) \equiv \operatorname{ln}(f(x))~(\bmod x^n)\)。

保证有 \([x^0]f(x) = 1\)。系数对 \(998244353\) 取模。

看到这里，大多数读者应该都是一脸懵：怎么对一个多项式求自然对数啊？

不要急，还记得前置知识吗？求导和积分。

前面还提到了一句话：把一些式子通过求导变成方便计算的形式，然后再用积分求出答案。

那么，做法已经呼之欲出了：

对 \(g(x)\) 求导有：\(g'(x) \equiv \operatorname{ln}'(f(x)) * f'(x) \equiv \frac{f'(x)}{f(x)}~(\bmod~x^n)\)。

这不就是求一个导数和一个逆元吗？

最后积分回去就好啦。

对于常数项，显然有 \(e^0 = 1\)，即 \(\ln 1 = 0\)，所以保证了 \([x^0]f(x) = 1\)，否则是很难做的。

多项式牛顿迭代

给出一个度数为 \(n - 1\) 的多项式 \(f(x)\)，求出一个度数同为 \(n - 1\) 多项式 \(g(x)\)，满足 \(f(g(x)) \equiv 0~(\bmod~x^n)\)。

（注意此处的 \(\bmod~x^n\) 会同时对 \(f(x)\) 和 \(g(x)\) 取模。）

保证有 \([x^0]f(x) = 0\)。系数对 \(998244353\) 取模。

首先需要知道前置知识：泰勒展开。

对于多项式 \(f(x)\)，如果我们已经知道了 \(f(x_0)\) 的值，那么 \(f(x)\) 的值可以用如下方法表示： \[ f(x) = \sum_{i = 0}^{\infty} (\frac{f^{(i)}(x_0)}{i!} * (x - x_0)^i) = f(x_0) + \sum_{i = 1}^{\infty} (\frac{f^{(i)}(x_0)}{i!} * (x - x_0)^i) \] 其中 \(f^{(i)}(x_0)\) 指的是多项式 \(f(x)\) 在 \(x_0\) 处的 \(i\) 阶导数，简单来讲就是对它求 \(i\) 次导数得出的值。

这个公式的证明和其它用法大可不必知道，我们只会用它做一些最基本的操作，不会超出我们的认知范围。

还是考虑倍增。

设求出了一个多项式 \(t(x)\)，满足 \(f(t(x)) \equiv 0~(\bmod~x^{\lceil\frac{n}{2}\rceil})\)。

那么我们要求多项式 \(g(x)\)，满足 \(f(g(x)) \equiv 0~(\bmod~x^{n})\)。

如果我们将 \(f(x)\) 在 \(t(x)\) 处进行泰勒展开，得到：

\[ f(g(x)) \equiv f(t(x)) + \sum_{i = 1}^{\infty} (\frac{f^{(i)}(t(x))}{i!} * (g(x) - t(x))^i) \equiv 0~(\bmod ~x^{n}) \]

又由于 \(f(g(x)) \equiv 0~(\bmod~x^{\lceil\frac{n}{2}\rceil})\)。

所以 \(g(x)\) 和 \(t(x)\) 的前 \(\lceil\frac{n}{2}\rceil\) 项相同，也就是说 \(g(x) - t(x) \equiv 0~(\bmod~x^{\lceil\frac{n}{2}\rceil})\)。

那么我们可以发现，在 \((g(x) - t(x))^i\) 这个式子中，当 \(i\ge 2\) 时，前 \(n\) 位必然为 \(0\)。

所以整个式子就等于： \[ f(g(x)) \equiv f(t(x)) + f'(t(x)) * (g(x) - t(x)) \equiv 0~(\bmod~x^{n}) \] 稍加推导一下就有： \[ g(x) = t(x) - \frac{f(t(x))}{f'(t(x))}~(\bmod~x^{n}) \] 这样就可以倍增求出 \(g(x)\) 了。时间复杂度 \(\mathcal{O}(n \log_2n)\)。

多项式指数函数

给出一个度数为 \(n - 1\) 的多项式 \(f(x)\)，求出一个度数同为 \(n - 1\) 多项式 \(g(x)\)，满足 \(g(x) \equiv e^{f(x)}~(\bmod x^n)\)。

保证有 \([x^0]f(x) = 0\)。系数对 \(998244353\) 取模。

显然把 \(e^{f(x)}\) 求导不会有任何变化。

所以，我们考虑用刚刚学到的牛顿迭代法做这道题。

我们构造出一个多项式（不管它的实际系数）\(h(g(x)) \equiv \operatorname{ln}(g(x)) - f(x)~(\bmod~x^n)\)。

那么把 \(g(x) \equiv e^{f(x)}~(\bmod x^n)\) 代入就有： \[ h(g(x)) \equiv \operatorname{ln}(g(x)) - f(x)\equiv\operatorname{ln}(e^{f(x)}) - f(x) \equiv f(x) - f(x)\equiv0~(\bmod~x^n) \] 然后我们就可以套用上面的多项式牛顿迭代了： \[ g(x) \equiv t(x) - \frac{\operatorname{ln}(t(x))-f(x)}{\frac{1}{t(x)}}\equiv t(x) - t(x) * (\operatorname{ln}(t(x)) - f(x))~(\bmod~x^{n}) \] 注意一点：\(f(x)\) 在求导的时候是被看作常数的。

对于常数项，可以参考上文《多项式对数函数》的证明。

多项式快速幂

给出一个度数为 \(n - 1\) 的多项式 \(f(x)\) 和一个正整数 \(k\)，其中 \(k \le 10^{10^5}\)，求出一个度数同为 \(n - 1\) 多项式 \(g(x)\)，满足 \(g(x) \equiv e^{f(x)}~(\bmod x^n)\)。

保证有 \([x^0]f(x) = 1\)。系数对 \(998244353\) 取模。

如果刷高中数学题刷得有感觉，应该可以一眼看出这里可以求 \(\ln\) 后求 \(\exp\)。

所以，我们求出 \(t(x) = \ln(f(x))\)，那么有 \(\ln(g(x)) = k * t(x)\)。

于是我们把 \(t(x)\) 的每一项的系数乘上一个 \(k\)，最后对 \(t(x)\) 求 \(\exp\)，得到的即为 \(g(x)\)。

时间复杂度 \(\mathcal{O}(n \log_2n)\)。

不过在贴代码前，有一个地方需要思考：\(k\) 在读入时应该对什么取模？

可能会有人脱口而出是 \(\varphi(998244353)\)。但是，需要郑重声明的是：欧拉定理在多项式的运算意义下是不成立的，正确的模数应当为 \(998244353\) 本身（阅读完下文后，可以发现这也是规定了 \([x^0]f(x) = 1\) 的原因）。

证明：

不失一般性地，我们考虑证明：\(f^p(x) \equiv f(x^p)~(\bmod~p)\)。

由二项式定理和 \(\operatorname{Lucas}\) 定理可以得出：\((a + b)^p \equiv a^p + b^p~(\bmod~p)\)。

考虑数学归纳法，由费马小定理可以得出，结论在度数为 \(0\) 的多项式中必然成立。

我们设 \(f(x)\) 度数为 \(k\) 且 \(f(x) = g(x) + a_kx^k\)，同时上述结论对于度数为 \(k - 1\) 的多项式成立。

则有 \(f^p(x) \equiv g^p(x) + a_k^px^{kp} \equiv g(x^p) + a_kx^{kp}\equiv f(x^p)~(\bmod~p)\)。

在这道题中，由于 \(n < p\)，又因为 \([x^0]f(x) = 1\)，所以在模 \(p\) 意义下有 \(f^p(x)\equiv f(x^p) \equiv 1~(\bmod~x^n)\)

因此，指数必须对 \(998244353\) 而非 \(\varphi(998244353)\) 取模。
